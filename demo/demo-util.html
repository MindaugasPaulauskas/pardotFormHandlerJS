Util Demo: Utility functions

This demo is a showcase of Utility functions.

pardotFormHandlerJS.getFormValues()

Copy below code to your console and check the results:

let forms = document.getElementsByTagName("form");
for (let i = 0; i < forms.length; i++) {
    console.log(forms[i].className, ":");
    console.log(pardotFormHandlerJS.getFormValues(forms[i]));
}
